# marcusthelegend


Landing page lists worlds with CRUD options.  A world has a description and an image generated from the description.  After the writer enters a description there will be a choice:  generate image or add image;

A world is made up of stories which have a description and an image generated from the description .

A story is a sequence
  - images with descriptions (writer enters description. nano2 creates images which are saved.
  - narrative pages (entered by writer)

Worlds and stories are persisted in the database.
